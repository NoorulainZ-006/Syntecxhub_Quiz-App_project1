# 🚀 QuizMaster

**QuizMaster** is a modern, responsive, and interactive web application developed as part of the **Syntecxhub_Quiz-App_project1**. It features a sleek **Glassmorphism** interface and fluid animations to provide a premium user experience.

---

## ✨ Core Features

* **Immersive Visuals:** Clean design with animated background glows and mesh gradients that eliminate empty screen space.
* **Glassmorphism UI:** Translucent cards and sidebars with high-quality backdrop-blur effects for a futuristic look.
* **Interactive Feedback:**
    * **Vibration Effect:** Visual shake animation triggered when a user proceeds without selecting an option.
    * **Instant Highlighting:** Immediate color feedback for correct and incorrect choices.
* **Smooth Transitions:** Powered by **GSAP** for high-performance screen and card movements.
* **Multi-Category Engine:** Comprehensive quizzes covering:
    * Web Development (HTML, CSS, JS, React)
    * Artificial Intelligence & Machine Learning
    * C++ Programming & OOPs
    * Cybersecurity & Ethical Hacking
    * General Knowledge (Friendship Trivia)
* **Smart Navigation:** Custom-animated hamburger menu providing seamless accessibility on mobile devices.
* **Final Analytics:** A detailed post-quiz summary screen showing performance and correct answers.
